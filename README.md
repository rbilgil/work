**Your product builds itself while you sleep.**

Whirl is an autonomous product development platform that unifies your team's discussions, decisions, and context to empower AI agents to ship work 24/7.

## Key Features

- **Context as Infrastructure:** Turn your specs, Slack threads, and documentation into a structured workspace for AI agents.
- **Mission-Driven Workflow:** Queue up high-level objectives and let Whirl handle the orchestration.
- **Autonomous Execution:** Agents work through the night, weekends, and vacations‚Äîno babysitting required.
- **Seamless Review:** Wake up to completed work, fresh PRs, and resolved tickets waiting for your approval.

## Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion
- **Backend:** Convex
- **Authentication:** Clerk
- **UI Components:** Lucide Icons, Radix UI

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
